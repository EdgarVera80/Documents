------------------ CURSO LINUX ------------------
El  pipe "|" sirver para concatenar la salida de un comando y convertirlo en entrada de otro. ejemplo:
>ls - la /tmp | more

Comando grep filtra cadenas de caracteres.
>ls -la | grep xyc
>grep zeus /etc/host  "Buscar la palabra zeus dentreo del archivo host"
>ls - la *.gz | grep xzy "Busca todos los archivo que tenga en el nombre xyz y que su extención gz"

Comando man + comando "muestra las ayudas del comando especificado"
Comando cat: muestra el contenido de un archivo.
Comando head: muestra las primeras líneas de un archivo con el modificador  -n 200
Comnado tail:
Comnado strings: busca textos dentro de un archivo binario
Comando which: Sirve para saber si un comando está instalado.
Comando who: muestra que usuarios está conectados al sistema operativo y por que medio está conectados (consola, ssh, etc.)
Comando lsof: muestra los archivo abierto a partir de un directorio.

Check list para servisar un servidor:
Comando df: /dev/ "la carpta dev se almacenan la lista de dispositovos"
Comando du - h /* "lista la cantidad en megas o gigas de las carpta que están de bajo de la raíz"
Comando du - hs /* "lista la cantidad en megas o gigas haciendo la suma del tamaño por carpetas"
Comando free "Muestra información sobre la memoria"
Comando top "Muestra una lista de proceso que se están ejecutando en tiempo real"
Comando ps "Lista los proceso del sistema"
Coamndo service ssh status "Muestra el status del servicio ssh", modificadores [status, start, stop]
